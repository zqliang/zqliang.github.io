Projects

A sample collection.

CourseTutor


A responsive MEAN stack application for connecting students with tutors. Once signed in, users are able to search through courses, post comments, message other users, and find contacts.


The app was tested using Mocha, Chai, Casper and Phantom. Performance anaylsis was done with the use of YSlow.


NY Times API


Designed and documented a REST API that parses NY Times data provided in JSON format. A sample call to GET THUMBNAILS would retrieve img URLs and display them as a list of thumbnails using AJAX.


Server was set up with Node.js.

3D Stem Plot


A new feature for matplotlib written in Python, which allows users to generate stem plots in a 3D view and rotate the graphs by specifying a fixed axis.


A detailed sprint plan was followed and image tests were generated throughout the development of this feature.

Zotero


A batch tag editing plugin for the Zotero software which allows users to search for and edit multiple tagged resources in their library concurrently.


Black Hole Game



An HTML5 canvas game where the user must click on randomly generated black and blue holes to detroy them, before they pull in all the floating objects in space or before time runs out.


Levels progress in difficulty with increasing black hole spawns.
